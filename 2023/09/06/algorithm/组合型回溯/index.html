回溯

发表于2023-09-06|更新于2023-09-06
|阅读量:

回溯问题就是dfs问题，不过要加上恢复现场。dfs问题就可以转换为树的问题。包括两种：枚举所有值(for) 和 选和不选问题。分别对应多叉树和二叉树。

恢复不恢复现场：
有的代码 path 长度会预先分配好，这种情况下不需要恢复现场，因为会直接覆盖已经填入的数据。如果 path 初始化成空列表的话，会不断往 path 末尾添加数据，这种情况是需要恢复现场的，如果不恢复现场，path 就会越来越长，不满足要求。

301. 删除无效的括号

这是一道经典的选和不选方式做的组合型回溯问题。牢记只要有恢复现场的步骤，其他地方都要恢复现场，就是这里的选字母的时候。因为恢复现场这个过程就是对应树往上走的过程
